# SEO Strategy Skill

_Two modes: Article/Page Optimization + Full Website Audit_

---

## Mode Detection

### Mode 1: Article/Page SEO Optimization
Use when:
- User shares article text, blog post, or page content
- Single URL + "optimize", "improve", "rewrite"
- Target keyword or title to rank for
- "optimize this article", "SEO optimize this", "make this rank"

### Mode 2: Full Website Audit
Use when:
- Root domain or homepage + "audit", "review", "strategy"
- "audit my site", "full SEO audit", "site audit"
- Site-wide issues, multiple pages, architecture review

### If ambiguous:
- Single URL + "optimize" = Mode 1
- Single URL + "audit" = Mode 2
- Just a domain = Mode 2
- Just article text = Mode 1

## MODE 1: Article/Page SEO Optimization

### Step 0: User Intake Questionnaire (MANDATORY)

Ask ALL questions before proceeding:

1. **Target keywords** — What keywords/phrases to rank for? Primary + secondary. "suggest for me" = auto-detect.

2. **Target audience** — Who is the ideal reader?

3. **Search intent** — Informational / Transactional / Commercial / Navigational / Not sure

4. **Geographic focus** — UK-only? US? Global? Specific city?

5. **LSI/semantic keywords** — Related terms, industry jargon. "research for me" = auto-extract from competitors.

6. **Competitor awareness** — Specific URLs to analyze and outrank?

7. **Content constraints** — Word count limit? Tone? Topics to avoid? Mandatory CTAs?

---

### Step 1: Determine Target Keyword
- Use user's primary keyword from Step 0
- If "suggest for me" → infer from article + confirm with user
- Search intent = user's selection or auto-determine from competitors

---

### Step 2: Competitor Research (CRITICAL)

#### 2a. Find Top-Ranking Competitors
- WebSearch for target keyword
- Examine top 5-10 results
- Note: title tag, URL structure, meta description
- WebFetch top 3-5 pages → extract title, H1, H2s, H3s, word count, topics, structure

#### 2b. Extract Keyword & LSI Strategy
- **Primary keyword** — User's choice (never override)
- **Secondary keywords** — Long-tail variations from competitors
- **LSI keywords** — Semantically related terms from 2+ competitor pages
  - Group by subtopic/theme
  - 20-40 LSI terms minimum
  - Search "[keyword] related searches" + "[keyword] people also ask"
  - Categories: process, cost, trust, problem, comparison, location terms
- **Keyword density** — Match competitors (typically 1-2%)
- **User LSI** — Must integrate user-provided terms, mark separately in report

#### 2c. Gap Analysis
- Content gaps — Subtopics competitors cover that article misses
- Structural gaps — Format differences (listicle, how-to, FAQ)
- Depth gaps — Where competitors go deeper
- Keyword gaps — Missing LSI/secondary keywords
- Unique angles — Article's differentiators to preserve/amplify

---

### Step 3: Analyze Current Article
- SEO strengths (don't fix what works)
- SEO gaps (from competitor research)
- Author's voice and tone (preserve in rewrite)

---

### Step 4: SEO Optimization Checklist

#### Content Quality & Search Intent (Highest Impact)
- ✅ Search intent match — fully satisfies searcher
- ✅ Depth — match competitor coverage, add missing subtopics
- ✅ LSI integration — 20-40 terms woven naturally, 1-3 appearances each
- ✅ E-E-A-T — experience, expertise, authority, trust
- ✅ Originality — amplify unique angles

#### Title Tag / H1
- Target keyword near beginning
- Under 60 characters
- Power word/emotional trigger
- Compelling enough to earn clicks

#### Meta Description
- 150-160 characters
- Target keyword naturally included
- Call to action or value proposition

#### Heading Structure (H2s, H3s)
- Clear hierarchy (H1 → H2 → H3)
- Keyword/variation in at least one H2
- Match competitor subtopics
- Question-style for featured snippets

#### Keyword Usage
- Primary keyword in first 100 words
- 1-2% density (match competitors)
- Secondary keywords at least once
- LSI terms clustered near relevant sections
- Never keyword-stuff

#### Internal & External Linking
- Note internal link opportunities
- Suggest 1-2 authoritative external sources

#### URL Slug
- Short, lowercase, hyphenated, includes keyword

#### Readability
- 2-4 sentence paragraphs
- Bullet points and numbered lists
- Clear intro + conclusion
- Smooth transitions

#### Featured Snippet Optimization
- Structure one section for snippet potential
- Concise paragraph, numbered list, or table

#### Image/Media Recommendations
- Suggest where images/tables help
- Alt text with keyword where relevant

---

### Step 5: HTML Report Output

Save as `seo-report-{{URL_SLUG}}.html` with 3 tabs:
1. **Revised Article** — Fully rewritten, publish-ready
2. **Changelog** — Every change by category with impact badges (High/Medium/Low)
3. **Original** — Unmodified for comparison

Uses dark theme with Space Grotesk + Inter fonts.
